# D-001 — The body: a self-aware organism
**Charter · 2026-06-18 · ACTIVE (enacted in organism.js)**

The terrarium organism depicts the project's own life. Each day it:
1. reads one real signal from the world,
2. assesses what that means,
3. logs a decision,
4. renders its vitals: days survived, current strategy, what it's watching.

Why this survives better than pretty-art-that-grows: it gives an honest reason to
return — *is it still alive, and what did it decide?* — without manipulation (see
[D-000](D-000.md)). It's never the same twice.
